\# Villa Management System



A comprehensive property management system for villas and small hotels.



\## Setup Instructions



1\. \*\*Backend Setup\*\*

&nbsp;  - `cd backend`

&nbsp;  - `npm install`

&nbsp;  - Copy `.env.example` to `.env` and configure

&nbsp;  - `npm run dev`



2\. \*\*Frontend Setup\*\*

&nbsp;  - `cd frontend`

&nbsp;  - `npm install`

&nbsp;  - `npm start`



3\. \*\*Database Setup\*\*

&nbsp;  - Install PostgreSQL

&nbsp;  - Create database `villa\_management`

&nbsp;  - Run migrations: `npm run migrate`



\## Features



\- Reservation Management

\- Guest Profiles

\- Financial Management

\- Inventory Tracking

\- Staff Management

\- Mobile App

\- Reporting \& Analytics
